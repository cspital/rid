# rid

```bash
Generate some ids

Usage:
  rid [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  l           Ulid
  u           Uuid
  x           Xid

Flags:
  -h, --help      help for rid
  -n, --num int   number of ids to make (default 1)

Use "rid [command] --help" for more information about a command.
```
